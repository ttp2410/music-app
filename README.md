## MUSIC-APP

> Framework : next 13.4.2

#### Cài đặt

Clone source code:

```sh
git clone https://github.com/ttp2410/music-app.git
```

Cài đặt thư viện hỗ trợ:

```sh
npm install
//or
yarn
```

Chạy dự án:

```sh
npm run dev
//or
yarn dev
```

**Lưu ý**

-   Không được chỉnh sửa code trên branch main
